# Orizon – Catalogo viaggi sostenibili (Python)

Progetto realizzato da **Luca Rotundo** per il Master in Data Science, Analytics e Agenti AI.

## Descrizione
**Orizon** è una piattaforma immaginaria che propone esperienze di viaggio sostenibili, comunitarie e rigenerative.
Il progetto simula la gestione di un catalogo viaggi tramite un menù testuale interattivo.

## Funzionalità
- Visualizzazione di tutti i viaggi disponibili
- Aggiunta di un nuovo viaggio con controlli sugli input
- Ricerca semplice (match parziale su più campi)
- Filtraggio avanzato con criteri multipli
- Statistiche sul catalogo (medie, min/max, distribuzione per Paese, miglior rapporto impatto/costo)

## Struttura del progetto
- `Orizon.ipynb` → notebook principale con codice e menù interattivo

## Come eseguire
Apri il notebook `Orizon.ipynb` e avvia tutte le celle.
Il programma funziona in modalità interattiva tramite input da tastiera (menù testuale).

## Note
Scala impatto: **1–10**, dove **10 = massimo impatto positivo** su comunità/ambiente.
